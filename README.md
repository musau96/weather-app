# 🌦️ 🐺 wolfy Weather App

An animated weather application that provides real-time weather information with beautiful gradient animations.

## Features

✨ **Beautiful Animations**
- Smooth gradient background animation
- Fade-in effects for weather data
- Interactive button hover effects

🌍 **Weather Information**
- Search weather by city name
- Get weather for your current location
- Display temperature, wind speed, humidity, and "feels like" temperature

🔧 **Technology Stack**
- Pure HTML5
- CSS3 with animations
- Vanilla JavaScript
- Open-Meteo API (free, no API key required)

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name and click "Search" OR
3. Click "Use My Location" to get weather for your current location
4. View the current weather conditions with beautiful animations

## APIs Used

- **[Open-Meteo Geocoding API](https://geocoding-api.open-meteo.com/)** - For city coordinates
- **[Open-Meteo Weather API](https://open-meteo.com/)** - For weather data
